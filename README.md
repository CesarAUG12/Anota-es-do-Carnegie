Este projeto é um site de anotações desenvolvido para registrar e organizar frases e aprendizados do livro *Como Fazer Amigos e Influenciar Pessoas*, de Dale Carnegie.

O objetivo foi criar uma interface simples, intuitiva e agradável, permitindo visualizar facilmente os principais conceitos e reflexões do livro. O projeto também serviu como prática de desenvolvimento front-end, com foco em estruturação de layout, estilização e experiência do usuário.

Através deste projeto, pude reforçar conceitos de HTML, CSS e JavaScript, além de praticar a organização de conteúdo e criação de interfaces funcionais.

Link do projeto: https://frases-carnegie.netlify.app/
